# Engineering Aphorisms

- Reduce surface area. Increase leverage.
- More mechanism is more liability.
- Every option carries a cost.
- Repeated choice is a tax.
- One clear way beats many arguable ways.
- Prefer capability over ceremony.
- Prefer reuse over repetition.
- Performance is designed in, not patched in.
- Memory is part of performance.
- Waste memory, waste time.
- Quadratic cost needs a reason.
- Pay maintenance in instalments.
- Technical debt collects interest.
- If it is wrong, fix it.
- Nothing is sacred except correctness.
- Delay makes hard fixes harder.
- Transitional shims are tools, not homes.
- Build for the smallest sufficient team.
- Coordination cost is real cost.
- Every dependency imports risk.
- External dependencies also surrender control.
- Many may observe; few should own.
- Shared ownership should be rare.
- Wide use does not require wide ownership.
- Smooth production is a design goal.
- Friction is the enemy of velocity.
- Duplicated function is duplicated cost.
- Deliberate constraint is a productivity tool.
- Preserve optionality only when it earns its keep.
- Prefer broad compatibility over novelty.
- Newer is not better by default.
- Respect constrained targets.
- Do not make all targets pay for one target's needs.
- Abstract real structure, not ambition.
- Do not abstract for status.
- Do not force concrete where the model is abstract.
- Do not assume. Find out.
- Measure before moralising about cost.
- Overhead is only meaningful in context.
- A one-time cost is not a perpetual cost.
- Human-friendly and machine-friendly need not be the same.
- Retain control.
- Reduce variance.
- Pay costs where they are cheapest.
